# 📊 HR Analytics Dashboard

1. Short Description / Purpose

The HR Analytics Dashboard is a visually engaging and data-driven Power BI report created to explore employee attrition patterns and workforce insights within an organization.

The dashboard focuses on analyzing key HR metrics such as attrition rate, employee demographics, salary distribution, job satisfaction, and department-wise trends. This tool is intended for HR professionals, business analysts, recruiters, and decision-makers who want to identify workforce challenges and improve employee retention strategies.

---

2. 🛠 Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 Power BI Desktop – Main data visualization platform used for dashboard creation.
- 📂 Power Query – Used for data cleaning, transformation, and preprocessing.
- 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, and interactive visuals.
- 📝 Data Modeling – Relationships created between HR datasets for effective filtering and aggregation.
- 📁 File Format – .pbix for development and .png for dashboard previews.

---

3. 📂 Data Source

Source: HR Employee Attrition Dataset (Sample HR Analytics Dataset)

The dataset contains employee-related information such as:
- Employee demographics
- Salary details
- Department information
- Job satisfaction ratings
- Attrition status
- Years at company
- Education background

4. ✨ Features / Highlights

## • Business Problem

Employee attrition is one of the major challenges faced by organizations as it directly impacts productivity, hiring costs, and business performance.

HR teams often struggle to quickly identify:
- Which departments have higher attrition rates
- Which employee age groups are leaving most frequently
- Whether salary impacts employee retention
- How job satisfaction relates to attrition

---

• Goal of the Dashboard

To develop an interactive HR analytics solution that:

- Helps organizations monitor employee attrition trends
- Enables department-wise workforce analysis
- Identifies factors affecting employee retention
- Supports HR decision-making through data-driven insights

---

5. 📈 Dashboard Insights
